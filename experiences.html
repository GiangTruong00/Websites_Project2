Out-of-work Experiences

ROM hacking is the process of modifying a ROM image or ROM file of a video game to alter the game's graphics, dialogue, levels, gameplay, and/or other elements. This is usually done by technically inclined video game fans to improve an old game, as a creative outlet, or to make new, unofficial games using the old game's engine. ROM hacks either re-design a game for new, fun gameplay while keeping most of the items the same, or unlock/reimplement features that exist in the game's code but are not utilized in-game.

I have been an Assistant Mapper in several Pokemon games.
